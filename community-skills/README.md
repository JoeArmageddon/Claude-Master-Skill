# Community skills

Skills contributed by the claude-session-master community.

Install any skill:
```bash
mkdir -p ~/.claude/skills/[skill-name]
cp community-skills/[skill-name]/SKILL.md ~/.claude/skills/[skill-name]/SKILL.md
```

Or run `/new-skill` inside Claude Code to create your own.

## Contributing a skill

See [CONTRIBUTING.md](../CONTRIBUTING.md) for the full process.

Short version:
1. Create `community-skills/[your-skill-name]/SKILL.md`
2. Add a row to the index table above
3. Open a PR with title: `skill: [skill-name] — [one sentence description]`
